# ICC1 - Computer Science Fundamentals (Rust Edition) 🦀 ⚙️

This repository contains my solutions for the **Introduction to Computer Science I (ICC1)** course at the **University of São Paulo (USP - ICMC)**. 

While the course traditionally uses C, I've chosen to implement these fundamentals in **Rust** to master ownership, borrowing, and memory safety from the ground up.

## 🎯 Objectives
* Master low-level concepts (pointers, stack vs. heap, data structures).
* Apply Rust's unique memory management to classic CS problems.
* Build a solid foundation for High-Performance Computing and MLOps.

## 🛠️ Tech Stack
* **Language:** Rust 1.75+
* **Environment:** USP São Carlos - ICMC (Data Science Undergraduate)
* **Concepts:** Algorithms, Sorting, Data Structures, and Memory Safety.

## 🚀 How to Run
1. Ensure you have `rustc` and `cargo` installed.
2. Clone the repo: `git clone https://github.com/kaioken13/icc1-rust-solutions`
3. Run a specific exercise: `cargo run --bin exercise_name`
